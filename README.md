\# Zecpath AI Core System



\## Overview

Zecpath is an AI-driven recruitment platform that automates candidate screening using Resume Parsing, Voice Intelligence, and Behavioral Analysis.



\## Project Structure

\- `data/` - Raw resumes and logs.

\- `ats\_engine/` - Resume parsing logic (NLP).

\- `interview\_ai/` - Voice \& Video processing logic.

\- `utils/` - Helper functions (Logger, Config).

\- `tests/` - Unit tests for system stability.



\## Setup Instructions

1\. Create a virtual environment: `python -m venv venv`

2\. Activate it: `venv\\Scripts\\activate`

3\. Install dependencies: `pip install -r requirements.txt`

4\. Run setup check: `python tests/test\_setup.py`
