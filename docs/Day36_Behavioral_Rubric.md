# Behavioral & Psychological Evaluation – Zecpath AI

## Objective
To mathematically evaluate a candidate's grace under pressure by analyzing vocal delivery, emotional sentiment, and psychological confidence indicators.

## The Behavioral Scoring Formula
**Behavioral Score** = `(Delivery Confidence × 0.7) + (Positive Sentiment × 0.3) − Contradiction Penalty`

## Metric Definitions
| Metric | Description | How it is calculated |
| :--- | :--- | :--- |
| **Hesitation** | Stuttering / Crutch words | Penalizes excessive use of "um", "uh", "ah". |
| **Uncertainty** | Self-Doubt | Scans for nervous phrases like "I guess", "maybe", "not sure". |
| **Pacing (WPS)** | Panic / Freezing | Calculates Words Per Second. Ideal pacing is 1.5 to 3.2 WPS. Outside this range indicates rushing or freezing. |
| **Sentiment** | Attitude | Compares ratio of positive action words vs negative complaint words. |
| **Contradiction** | Backpedaling | Applies a severe 20% penalty if the candidate changes their answer mid-sentence (e.g., "Wait, no"). |
